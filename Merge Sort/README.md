## About The Project

Implementation of the recursive or iterative Mergesort with optimizations:

   * Using Insertsort on small parts of the array;
   * Do not performing the merge operation if the collection is already sorted;
   * Eliminating the copy to an auxiliary array;
   * Taking advantage of pre-sorted sections.
